Bem-vindo ao meu portfólio! Aqui você encontrará alguns dos projetos que desenvolvi utilizando tecnologias como HTML, CSS e JavaScript puro para garantir alta performance.

## 🌐 Acesse o Portfólio

Você pode acessar meu portfólio completo [clicando aqui](https://portfoliofmg.netlify.app/)

## 🚀 Tecnologias

Este portfólio foi desenvolvido utilizando as seguintes tecnologias:

- **JavaScript Puro:** Lógica de funcionamento sem dependências externas, focado em performance.
- **HTML5 Semântico:** Estruturação de conteúdo com tags semânticas para melhorar a acessibilidade e SEO.
- **CSS com Flexbox e Grid:** Layouts responsivos e eficientes, utilizando Flexbox para alinhamento e distribuição de espaço, e Grid para criação de estruturas complexas.
- **ScrollReveal:** Biblioteca utilizada para animações de carregamento suave dos elementos da página.

## 📚 Conceitos Aplicados

Durante o desenvolvimento deste portfólio, os seguintes conceitos foram aplicados:

- Desenvolvimento Front-End com foco em performance e usabilidade.
- Estruturação semântica com HTML5 para melhor acessibilidade e SEO.
- Layouts responsivos utilizando Flexbox e Grid, garantindo adaptabilidade a diferentes dispositivos.
- Implementação de animações de carregamento de elementos com ScrollReveal, melhorando a experiência do usuário.
